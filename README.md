BootFlask
=========

A simple tool for turn your flask projects more quick and fun..

Version:
----

0.1


Installation:
--------------

```sh
pip install BootFlask
```

or

```sh
pip install https://github.com/riquellopes/boot-flask/tarball/master
```

CookBook:
-------------
```sh
bootflask -p HelloWold
```
